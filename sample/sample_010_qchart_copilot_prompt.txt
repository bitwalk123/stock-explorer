プロンプト: PySide6 QChart リアルタイムトレンドチャート

目的:
PySide6 の QChart を使用し、リアルタイムでデータをプロットするトレンドチャートのサンプルコードを作成する。

要件:
以下の要件をすべて満たすこと。

1. フレームワーク: PySide6 (PyPIで入手可能な最新安定版) を使用する。
2.横軸 (X軸):
- 0 から 100 で範囲を固定する。
3. 縦軸 (Y軸):
- プロットされるデータに合わせてスケールを動的に変更する。
- 10刻みの切りの良いティックラベルを表示する。
4. データ生成とプロット:
- 0 から 100 までのランダムな値を生成する。
- QTimer を使用し、1秒ごとに1つずつデータを追加（プロット）する。
- データ追加回数が100回を超えたら QTimer を停止する。
- 1データ描画あたりの処理時間（ベンチマーク）をミリ秒単位でコンソールに出力する。
5. 固定ラインの表示:
- y = 60 の横線を、横軸の全範囲（0から100）にわたってプロットする。
- この線は赤色で表示する。
6. 描画品質とアニメーション:
- QChartView で QPainter.RenderHint.Antialiasing を設定し、描画を滑らかにする。
- QChart で QChart.AnimationOption.SeriesAnimations を設定し、データ追加時にアニメーションを有効にする。
7. APIの利用:
- PySide6 の最新の推奨される API (series.attachAxis(), points()) を使用し、非推奨の警告が出ないようにする。
